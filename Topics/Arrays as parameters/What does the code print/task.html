Here is a method that takes an array:

public static void method(int[] array) {
    if (array.length >= 3) {
        array[0] = 1;
        array[1] = 2;
        array[2] = 3;
    }
}


We invoke this method inside another one:



int[] numbers = { 4, 5, 6 };
        
method(numbers);
        
System.out.println(Arrays.toString(numbers));
